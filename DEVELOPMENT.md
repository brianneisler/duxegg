# Development

# Requirements

* node.js 6.0+
* npm 3.0+

*NOTE:* we recommend using nvm
```sh
brew install nvm
nvm install 8.5
nvm use 8.5
```

## Setup

```sh
git clone https://github.com/brianneisler/duxegg.git
cd duxegg
npm install
```

## Build

To build duxegg

```sh
npm run build
```

## Watch

To watch duxegg and build on every file change

```sh
npm run watch
```
